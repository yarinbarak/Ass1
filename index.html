Same query in Web1 and Web3.

Web1

Preconditions: User had a web page open and a query that he needs to get an answer for.
Action: The user is going over the page, and looks for a word or a sentence that seem to be close for what he looks for
The user finds something that answers his query and go out of the web page.
postAction: The user has the answer to his query but he spent too much time and effort.

Web3

Preconditions: User had a web page open and a query that he needs to get an answer for.
Action: The user types the query to a search engine and it returns the best suiting results for his query.
The user go over the results and find what he was looking for.
postAction: The user has the answer to his query, he didn't have to spend too much time or effort. It was fast and he just needed to type his query.
